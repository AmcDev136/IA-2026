# Red Neuronal para pasar de año
### Describir como crear una red neuronal para pasar de año, únicamente debe decir "si" si pasa de año o "no" si no pasa de año

Para crear una red neuronal para saber si un alumno pasa de año, se necesitan entradas como:
+ Calificaciones de cada materia
+ Promedio final.
+ Número de materias reprobadas.
+ Numero de faltas.
+ Resultados de los exámenes.
+ Si existen, puntos extras.

Entrenamos la red con un dataset supervisado con datos de alumnos de años anteriores.
